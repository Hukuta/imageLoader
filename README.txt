Программа скачивания картинок для магазина на CMS php5shop.
Ссылка на первую версию http://yadi.sk/d/BlrxPyXtBLEBK

Приминение:
У Вас магазин на движке php5shop и прайс с огромным количеством изображений (тысячи, десятки или сотни тысяч),
что сайт не справляется с обработкой такого количества изображений одним прайсом.
Данная программа поможет выполнить скачивание и обработку фотографий на Вашем локальном компьютере,
а затем Вы сможите загрузить готовые фотографии по FTP или FTPS протоколу.

Написана на языке программирования python с приминением
графического пользовательского интерфейса GTK+ (PyGTK).

Инструкия по использованию:
* Замените стандартный файл watermark.png своим, если Вы его создавали для Вашего магазина
* Запустить скрипт imageLoader.py (в Windows - двойным кликом по этому файлу)
* Если окно программы не открылось, см. инструкцию по установке.
* Если окно открылось, нажмите на кнопку 'Выбрать прайс' и выбирите файл xls прайса,
    который Вы заполнили для импорта товаров в магазин на php5shop CMS.
* Если это не первый запуск программы с этим прайсом, вы можете установить чекбокс 'Пропускать скачанные раньше',
    тогда не будут скачиваться изображения, файлы которых уже есть в каталоге images
* Нажмите на кнопку 'Скачать все картинки на этот компьютер' и ожидайте пока не появится диалоговое окно.
    Если прогрессбар не двигается и ничего долго не происходит,
    можно посмотреть сообщения в консоли и отправить их разработчику на анализ для исправления программы.
* После завершения работы загрузчика, в каталоге images появятся файлы изображений с защитным знаком
    и каталог small с маленькими копиями изображений. Воспользуйтесь удобным Вам FTP-клиентом для загрузки содержимого
    каталога images в каталог images/products/ вашего магазина.


Инструкция по установке в Windows:
Скачать все дополнительные программы одним архивом http://yadi.sk/d/1djzNpQx5J0bS
Для работы программы Вам необходимо скачать и установить соблюдая последовательность:
* Python 2.7:
Для 32-битных систем python-2.7.5.msi http://yadi.sk/d/bSZaQ6BM5Iuhq
Для 64-битных - python-2.7.5.amd64.msi http://yadi.sk/d/pxEjOaE85IufG
* PyGTK pygtk-all-in-one-2.24.0.win32-py2.7.msi http://yadi.sk/d/7-sm7coq5IwqX
* Python Imaging Library (PIL) PIL-1.1.7.win32-py2.7.exe http://yadi.sk/d/AiI37sHq5IwsZ

Инструкция по установке в Linux:
* Запустить программу в консоли, в случае появления сообщений об ошибках - установить недостающие модули
